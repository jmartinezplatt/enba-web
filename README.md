# Espacio Náutico Buenos Aires

Sitio web de Espacio Náutico Buenos Aires — travesías, escuela náutica, broker y servicios desde Puerto Norte, Buenos Aires.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Desarrollo local

```sh
npm install
npm run dev
```

El servidor arranca en `http://localhost:8080`.

## Build

```sh
npm run build
```

## Deploy

El sitio se deploya automáticamente en **Cloudflare Pages** con cada push a `main`.
