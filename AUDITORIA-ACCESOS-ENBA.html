  • Zone: DNS: Edit (para el dominio del sitio)

3. Anotar el Account ID (visible en Overview de cualquier dominio)
4. Configurar como GitHub Secrets:
CLOUDFLARE_API_TOKEN=el-token-generado
CLOUDFLARE_ACCOUNT_ID=tu-account-id
5. Instalar Wrangler en el proyecto: npm install -D wrangler
6. Crear wrangler.toml:
name = "enba-web"
compatibility_date = "2026-03-06"
pages_build_output_dir = "dist"
7. (Alternativa) Vincular repo en Cloudflare Pages dashboard → conectar GitHub → seleccionar repo enba-web → build command: npm run build → output: dist
